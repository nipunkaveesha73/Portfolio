# Nipun Kaveesha's Portfolio

This is my personal portfolio website, built with Next.js and React. It showcases my web development projects and skills.

## About

I'm a web developer passionate about creating modern, responsive websites and applications.

## Features

*   Interactive UI elements
*   Showcase of web development projects

## Technologies

*   Next.js
*   React
*   Tailwind CSS

## Get Started Locally

1.  Clone the repo:
    ```bash
    git clone https://github.com/nipunkaveesha73/Portfolio.git
    ```
2.  Navigate to the project:
    ```bash
    cd Portfolio
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```
4.  Run the development server:
    ```bash
    npm run dev
    ```
